Are you ready for the

slack down?

Issue

Main issues

We chose to focus on was the primary action button (hamburger menu) presently located in the top left corner. For a right-handed user, this is the least accessible area on the entire screen.
Both SEARCH and PROFILE buttons are also difficult to reach, they can be found in the top right corner.


Smaller issues

The messaging functions and how they are displayed in the default setting.

Notifications in workspace A are not displayed if you are working in workspace B.

Motivation


Steven Hoober: The Thumb Zone

After over 1300 observations he concluded that 49%
of the users use a one-hand grip, using only the thumbzone.


Dan Brown: Information Architecture Principles

#5: The principle of front doors

Give the user several paths to the section you want them to reach.

#6: The principle of multiple classification

The goal is to make the primary actions more accessible to the user.

Solution

Bringing this function out from the hamburger menu and down to the action menu makes the Workspace more accessible.

If you’re in workspace A there is no notification for workspace B. When pressing the workspace button in the action menu all workspaces will expand upwards on the screen, displaying notifications.

Just like the Group-button, # was also hidden under the hamburger menu, now one click less away. At first we were considering separating #, threads and private messaging into separate functions but we decided not to reinvent too much, changing things that are not really a problem.

Although this is a key function of the Slack service it doesn’t need to be displayed straight away. We hid the messaging function under WRITE. This function brings forth the keyboard and messaging functions, basically as they are today.

Bringing down the search function to the action menu, gathering all function in the action menu.

Even though the profile section is not very accessible in the bottom right corner of the action menu it is not one of the primary action buttons. Also, we gather everything in the action menu at the bottom instead of having some action at the top and at the bottom.
