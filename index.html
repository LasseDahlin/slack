Are you ready for the

slack down?

Issue

  • The main issue we chose to focus on was the primary action button (hamburger menu) and how it is placed in the top left corner. For a right-handed user, this is the least accessible area on the entire screen. The search and profile functions were also placed in the header of the screen.
  • We also address smaller issues such as the messaging functions and how they are displayed as soon as the app/page is started. Could it be prompted later or in a different way?
  • Notifications in workspace A are not displayed if you are working in workspace B. Could this be done differently?
  • Although there is a slide function that lets you slide through the primary action we wanted to give these functions more attention.

Motivation

Steven Hoober coined “The Thumb Zone” back in 2011. Through over 1000 observations he concluded that 49% of the users use a one-hand grip while and 75% relied on their thumb when using a smartphone.

Although there is a sliding function letting the user pan through the primary functions of Workspaces and #’s, we found them by chance. Supporting this through a combination of Dan Brown’s principle 5, The principle of front doors and 6, The principle of multiple classification, we wanted to give the users different ways of reaching the primary actions of Slack.

Solution

Workspace

Bringing this function out from the hamburger menu and down to the action menu solves the trouble of reaching this primary function. This way the Workspace is more accessible and one less click away.

We also wanted to better highlight if there is a new message in another workspace. As things are now, if you’re in workspace A there is no notification for workspace B. When pressing the workspace button in the action menu all workspaces will expand upwards on the screen, displaying notifications.

#

Just like the Group-button, # was also hidden under the hamburger menu, now one click less away. At first we were considering separating #, threads and private messaging into separate functions but we decided not to reinvent too much, changing things that are not really a problem.

Write

Although this is a key function of the Slack service it doesn’t need to be displayed straight away. We hid the messaging function under WRITE. This function brings forth the keyboard and messaging functions, basically as they are today.

Search

Bringing down the search function to the action menu, gathering all function in the action menu.

Profile

Even though the profile section is not very accessible in the bottom right corner of the action menu it is not one of the primary action buttons. Also, we gather everything in the action menu at the bottom instead of having some action at the top and at the bottom.
